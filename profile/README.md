### What is Zutrix?

Zutrix is an all-in-one SEO platform trusted by **2,000+ professionals** worldwide. We help businesses track rankings, discover keyword opportunities, monitor backlinks, audit websites, and measure AI search visibility — all from a single dashboard.

### Open Source

We believe in building for the AI-native SEO workflow. These open-source projects connect Zutrix data to your favorite AI tools:

| Repository | Description |
|:-----------|:------------|
| **[zutrix-mcp-server](https://github.com/Zutrix-Technologies/zutrix-mcp-server)** | MCP Server — connect Claude, Cursor, VS Code, and other AI tools to your Zutrix SEO data |
| **[skills](https://github.com/Zutrix-Technologies/skills)** | Pre-built SEO workflows for AI assistants — audits, keyword research, rank tracking reports, and more |

### Quick Start

```bash
npm install -g zutrix-mcp-server
```

Then add to your AI tool's MCP config:

```json
{
  "mcpServers": {
    "zutrix": {
      "command": "npx",
      "args": ["-y", "zutrix-mcp-server"],
      "env": {
        "ZUTRIX_API_KEY": "your_api_key_here"
      }
    }
  }
}
```

Get your API key at [zutrix.com/dashboard/settings](https://zutrix.com/dashboard/settings).
